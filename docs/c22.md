泥水到了脚踝。

不是水，是泥。黄泥和稀释过的粪水混在一起，踩进去拔不出来，拔出来带出一串泡，泡破了，放个闷屁似的声音，释出一股甜腻的腥臭。六月中旬的新平，雨季来了，栅栏围子顶上的破草席全透了，雨点顺着席缝往下漏，一滴一滴落在人头顶，落在肩背，落在已经发馊的伤口上，人痒得想抓，抓破了更烂。

罗石躺在最靠边的角，背抵着拒马桩，桩子是松木的，剥皮没剥净，残留的树皮被雨泡胀，摸上去滑腻。他的僧袍早就没了，在进营第一日就被扒走换了一口黑糊糊。此刻只剩一层土黄色的中衣，早就看不出颜色，袖口烂成须，贴在小臂上，袖口里兜了一包泥浆。他想挪一挪，让背离桩子远点，泥吸住了他，他使一次劲，肋骨像要从皮下顶出来，失败了，重新陷回去。

烧起来是前天夜里的事。

起初是冷，关中六月的雨冷得不讲理，从后颈往脊梁里钻，他缩成一团，牙齿打颤，敲出细碎的咯嗒声。旁人听见了，以为他要死了，默默往旁边让出半尺空隙，生怕死气沾身。半夜过后冷过去了，转成烫，额头滚烫，手心却冰，胸口像压了一块烧红的烙铁，每吸一口气，烙铁就往肺里推一寸。他张嘴想喊点什么，舌根那枚硬结硌住，舌头抬不起来，嗓子里只剩一点漏风的嘶嘶。

量子信道的第三次粘膜钙化，他在心里给自己的症状打标签，像当年在北邙维度科技的地下室给小白鼠的舌头贴标签那样。熟练，冷静，精确，然后放任。

这一次排异来得更重。三十年，是物理极值，是陆志强在主控台上随手把三十分钟改成三十年时敲下的数字，也是当初白板上他亲手写的TTL——Time To Live，超过即丢包。今天是第十七年，粒子层面已经开始不认这个时空，生物体把自己当外源进程，发起oom-killer。CPU占用飙到百分之百，风扇狂转，内存泄漏，无人GC，watchdog迟早会把他kill掉，留下一行日志：process exited with code -9。

他闭上眼，准备让自己关机。

远处传来放饭的梆子声。每天辰时一梆，申时一梆，声音哑，敲的人没力气，梆子也敲得不情愿。营里动了一下，像一锅即将沉底的粥被筷子搅动，人影爬起来，扶着栅栏，扶着别人肩膀，往门口挪，去抢那一勺掺沙子的黑面糊。有人摔进泥里，手里还死死抠着木片碗，有人被踩了手，叫都不叫。看守提着木桶往地上泼，泼出来的糊不到半指深，褐色的，悬着麦麸和细沙，苍蝇落在上面，聚成黑压压一团。

罗石没动。他动不了，也不想动。他的那份，会有人抢走，这是这个营地的GC机制，自动回收无人认领的资源。

一双脚停在他面前。

很小的脚丫，套在一双过大、露出脚趾的草鞋里，鞋带是麻绳，麻绳染成黑色，分不清是泥还是血。脚趾甲全裂了，脚背上淌着雨水和脓水。更小的影子蹲下来，挡住一点雨。

“娘，就是他。”

是个小女孩的声音，七八岁，细，哑，像很久没喝过水，舌头粘在上颚上硬扯出来的。

罗石没睁眼。他烧得眼前全是光斑，光斑里有代码在跑，跑的是当年那份自毁脚本的注释：When the machine becomes a weapon, destroy it. 赵婷敲这行字时，手指卡了一下，问他要不要加句号。他说不用，破坏不需要句号。

一只更瘦的手碰了碰他肩头，怕惊动他，极轻，又极急切。

“大师。”女人的声音，抖得不成调，关中口音，混着哭腔，和雨声捣在一起，“大师，你看看我，你不记得我们了？”

罗石勉强睁开一条缝。雨水顺着睫毛往眼里灌，刺得疼。眼前是两个人，女人三十出头，瘦得颧骨要从皮里戳出来，头发用一根树枝绞住，身上披的不是衣裳，是一块拆下来的门帘，门帘上还有半个福字，倒的。身后牵的小女孩更小，脸脏得只有眼睛是亮的，眼窝深陷，却不浑，像这种地方很难得的清亮。女孩手里捧着半块黑饼，饼只有孩童手掌大，边缘全是手指掐出来的印，黑黄色，里面嵌着明显的沙砾和草屑，还沾着泥，泥是湿的，饼是被体温暖过的。

女人见他看过来，扑通跪进泥水里，膝盖砸起泥点，溅在女孩草鞋上。她双手合十，胡乱地拜，嘴里语无伦次。

“长安，五重寺门口，建元十四年冬天，城里头瘟疫，都说要封城烧人啦。我们一家三口住城西小巷，孩子爹烧得不省人事，我和粟儿也起不来。门口支了口大缸，说是圣水，喝了就能退烧。都去抢，都去抢，我们挤不进去。后来是您，您叫人把缸抬出来，一碗一碗舀给我们，还说一天一碗，连喝三天。孩子爹喝了第二天就出汗了，汗出透了就好了。我们好了以后，去寺里头要给你磕头，小沙弥说您已经去给军爷们看病去了，不在。我们一家子在佛前给您点了三盏灯，灯油是用卖了一根簪子换的。您不记得了吗？”

建元十四年。

四个字像一根针，刺破了他脑子里糊成一片的浆。公元378年，苻坚建元十四年，长安大疫。史书只有一句，人民流散，疫死者衆。可对于蹲在城里的人，那一句就是一座山。他记得那年冬天特别冷，五重寺香火少了一半，门外倒的人多了。他那时为了立稳神僧的人设，确实干过一件事。

他的布洛芬。那瓶从赵婷手里接过来的Costco大瓶装，白色塑料瓶，标签上印着蓝色字母，Ibuprofen 200mg/500 tablets，瓶身还有赵婷用马克笔写的Expiry 2025，字迹被手汗蹭花了。两瓶并成一瓶，哗啦哗啦像倒豆子，是高烧时他在潼关路上反复回想的画面。当时烧得烫手的孩子太多，他问寺里借了一个药臼，戴着从实验室带出来的、早就开线的防静电手套，把药片一粒一粒碾成粉。没有称，他就凭手感，一小撮粉兑一缸井水，搅拌，用木勺舀出来。寺里的小沙弥说大师这是佛家慈悲圣水，其实是极低浓度的现代解热镇痛剂悬浊液，剂量只能算安慰，可对于伤寒、流感引起的继发高烧，配合多喝热水，竟真能把许多人从阈值线下拽回来。分子层面的干预，不触发宏观坍缩，只改自由变量的生死。他那时没想这么多，只想让门外吵闹声小一点，让苻坚的太史令少在岁星分野上挑他毛病，说他妖言惑众，岁在析木，燕有灾，岁在星纪，秦当受困那套话术快用烂了，得补点真的功德。

他竟然救过眼前这两个人。

罗石的嘴唇动了动，想说我不记得。那种话卡在钙化的舌根上，说不出来。他看着女人，女人也在看他，眼泪混着雨水往下淌，流进她干裂的嘴角，她没擦，继续拜，像要把过去八年欠下的磕头全补回来。

“大师，您怎么会在这里？您不是在苻天王身边当国师吗？苻天王呢？”

苻坚死了。弓弦绞死在新平佛寺的槐树下，眼睛望着长安方向，眼角有泪，天上那只鸢飞得低，泥像还在后秦大殿里受香火，姚苌半夜磕得额头出血，喊着小羌怎配天子之位。这些话在罗石舌头上滚了一圈，没出来，化作一声含糊的呜咽。

小女孩把那半块饼往前递了递，手腕细得像一截枯枝，青筋突起，却尽量把饼举得平稳，不让它碰到泥水。

“娘说，”小女孩低声说，声音认真得不像这个年纪，“那年喝了大师的水，才活下来。现在大师病了，要吃东西。我吃过一口了，剩下的给你。”

罗石想摇头。他当过架构师，算过账。这营里每人每天一勺面糊，不到八十千卡，撑不过十天就会器官衰竭。人要把基础代谢压到极限才能活，半块掺沙饼，撑死一百千卡，对于瘦成这样的母女，省下来就是把自己的命往外掏。

女孩以为他嫌饼脏，急了，把饼掰得更碎一点，把混在饼面上最显眼的一粒沙砾用指甲剔掉。剔不干净，面团里全是沙，黑麦没磨细，麸皮刺手，里面还混着观音土，那种土吃多了会把肠子堵死，但它能把肚子填满，骗过大脑。她把稍干净的那一角掰得更小，塞进罗石手心。

饼是凉的，边缘被体温焐出一点点余热，余热转瞬被雨水夺走。罗石的手指扣不紧，饼差点滑回泥里，女孩用两只手托住他的手背，把那块饼硬压在他掌心，压得指节泛白。

“吃，”她命令他，又像是求他，“吃了就不烧了。”

罗石盯着那半块饼，指尖发抖。

当年那粒白色药片，布洛芬分子式C13H18O2，分子量206.28，常温下微溶于水，镇痛机理是抑制环氧化酶，减少前列腺素合成。对于十七岁的身体，那是一个外来变量，却把她们从已标记删除的数据集里捡回来，让她们活成自由变量，长成八年后的样子。八年后，自由变量带着一枚混着泥沙、草屑、可能还有霉菌孢子的碳水化合物聚合物回到他面前，想要把他的温度降下来。这一次没有200毫克的药效，只有碳、氢、氧、氮，加上一点关中黄土里硅酸盐颗粒，物理上几乎不可吸收，却在化学计量外多了一项东西，罗石过去十七年在邺城、长安、淝水、参合陂都没算进去的东西。

笃。

不知道谁在营里敲木板，规律一下，提醒放饭结束，也像他当年在邺城禅房里敲木鱼，一下一下敲在尸山血海里，敲在满地白骨的峡谷里，敲在槐树绞索收紧后那段诡异的寂静里。

他把饼塞进嘴里。没法嚼，舌根硬，牙齿松，泥沙硌得牙床出血，血混着面糊咽下去，咸的，涩的，土腥的，剌嗓子。他强行吞，喉结滚动一下，食道被粗糙的麦麸划得疼。吞下去后，咳出一口带泥的痰，落在自己胸口，黑色。

女孩看着他吞完，嘴角往上提了一点，笑，露出缺了一颗的门牙，门牙旁边的牙全黑了。她把剩下的小半块也递过去，整个动作藏在母亲门帘的阴影里，不让远处看守看见。小声说，明日还有，我再省。

田芸把他往自己搭的草棚下挪了一点，草棚是三根棍子支的，顶是一块破席，席子破洞能看见天。她从怀里掏出一块稍干点的破布，浸了点雨水，笨拙地擦他额头的泥。雨水本就是冷的，擦在烧烫的皮上，激起一层鸡皮疙瘩，罗石反而清醒一点。

清醒时，记忆不受控地流水线一样过。

最先是太阳能板，在潼关口被流矢划破铝边框，MPPT排线断裂，彻底黑屏那一下，屏幕最后定格是相册，Costco通道里赵婷推两辆车的照片。板子是他和刘教授反复算过的功率冗余，为了在369年充手机，现在成了废铁，埋在路边，像墓碑。
然后是充电线拉扯，那根USB-C线，他在北邙地下室准备区和陆志强掰扯，说手机算一样，板子算一样，线也要算一样？陆志强拎起来晃晃笑，说罗子你怎么这么计较。当时杀人的羞辱感，现在想起来竟像彼岸的灯，提醒他来处。
接着是三千万，姚雪梅那份审计报告，红章，公章，法务照片，发朋友圈说创业的弟弟，其实连小区门口烧烤摊都不愿意看，开曼空壳BVI套开曼，开曼套维京，五层SPV嵌套，标准教科书式洗钱结构，傻子才信是创业。那本是他启动自诊断的触发器，却在舱门关闭后才想明白。
接着是金刀。苻坚临死前提起的那把金刀，王猛当年送去邺城借刀杀人的信物，信上说慕容垂不可留，以金刀为信，令其自裁。苻坚又想立圣王人设，又舍不得杀，托人放水，酿出今天后秦大营里这么多无辜百姓。金刀没有刀锋，杀人不见血，最毒。
接着是假牙，太极殿接风宴上王猛发难，问漏风的帝王相准不准，罗石急中生智抬出史书冷门细节，慕容垂坠马断齿，安的木假牙，取下来当场漏风，满殿大笑，躲过一劫。那一笑也埋下日后监视的种子，王猛盯他像盯妖僧。

全是细节，像碎纸机里被切碎又被强行拼起来的合同，每个毛边都对得上。

田芸还在擦，动作轻，怕弄疼他。阿粟蹲在一旁，用树枝在泥里画圈，圈里画个小人，小人没有脸，只有个光头，像他。画完抬头问娘，我们以后还能回长安吗？田芸没回答，把门帘往女儿身上多盖了一点。

罗石的烧没有立刻退，夜里更重。雨停了，风起来，草棚被吹得呼呼响，破席掀开一角，露出半块被墨色浸透的夜幕，夜幕里没有星，只有远处后秦军营火把一明一灭。他蜷在泥里，胸口那半块泥沙饼像一块石头，硌在胃里，却慢慢化开，化成一点点热，热顺着静脉往四肢流，流到指尖，指尖稍稍有了知觉，能攥住阿粟递过来的草棍。

笃。

他在心里敲了一下木鱼，没有木鱼，只有舌根硬结与牙齿磕碰的声音。

笃。

第二日辰时梆子响，阿粟又来了，这次更快，手心藏着比昨日更小的一角饼，饼里泥沙更多，掺了野菜，野菜是灰灰菜，苦的，放在嘴里能把胆汁都激出来。她像做贼一样塞进他手里，塞完立刻跑开，去排队领自己的那勺糊，糊领回来了，她当着罗石面喝一半，剩下倒在破瓦片上，用手指蘸得干净，然后把瓦片底下沉淀的那点稠的，刮给罗石。

罗石没推辞。他知道推辞比吃下更残忍。他用被泥糊住的手指把那点糊抹进嘴，沙砾在牙齿间嘎吱响，响得耳朵疼。他看着女孩蹲在泥里吃手指的样子，眼眶酸了一下，酸得发胀，却没有水，烧把泪早蒸干了。

吃完，阿粟问他，你叫什么名字呀，大师都叫你大师，你总有名字吧？

罗石张张嘴，发不出完整的音。他想起自己原名罗石，一块石头，北邙山防空洞里，陆志强叫他罗子，赵婷叫他师兄，姚雪梅叫他阿石，走到哪里都是石头，冷的，硬的，在实验室恒温恒湿里当性能件，在战场上当看客的石头。石头旁边若是有个人，能暖一暖，能把这漫长的一夜熬过去，但过去十七年他没添，在邺城没添，在长安太极殿论岁星分野时没添，在淝水尸堆里敲木鱼时也没添，现在泥水泡到脖子，倒是被人递了一块饼，饼里有一点点人味。

他伸出手指，沾了点泥水和自己嘴角带血的涎，在女孩瘦削的手心写字。先写罗，再写石，笔画像刀，划得她手心痒，她咯咯笑，又忍住，不敢大声。写完石字，他顿了很久，泥水在指尖要干了，血也凝住，女孩的掌纹里全是细小的泥沙，像河道分叉。

他没有擤鼻涕，也没有宣佛号，只是看着那手心，那两笔，像看着自己十七年的代码仓库，最后一次commit前，犹豫要不要改一个变量名。

笃。

雨又要来了，云从长安方向压过来，黑压压。营外万人坑的乌鸦起了一下，又落下，走不动。

那一笔人字旁还没有添下去，时机还没到，但已在指尖。
