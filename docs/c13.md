---

长安的雪化得晚。

雪水沿着五重寺的瓦当往下滴，滴在廊下的旧蒲团上。蒲团已经坐出了人形，中间凹下去一块。罗石坐在凹陷里，腿麻了。

寺门外时常有车马停下。

丞相府的人来了，说景略公请大师过去辩一辩《穆天子传》里西域三十六国的风土。又一次是问关中今年的租庸怎么算，三等户和四等户的绢到底差几匹。佛理一句不问。

罗石知道这是在扫盘。

像 Code Review 时逐行过一遍 git blame。每一个回答都要和上一个 commit 自洽，不能有 merge conflict。

王猛坐在主位，清癯，穿一身半旧的青袍，手里捏着一只陶盏，热气缭在鼻尖。他不信佛，但问佛经问得细。问到《阿含》里“石旁”是什么意思，问到僧人为什么不许私蓄金银，问到天竺的僧人怎么过冬。

罗石答得滴水不漏。

他不敢答得太笨。笨了，王猛会直接判定为装傻，当场丢进只读黑名单。也不敢答得太聪明。太聪明，那个在长安太极殿上靠着手机里背下来的《晋书》冷门条目才躲过“帝王相”死题的番僧，就成了活的威胁模型。

有一次王猛忽然问：“大师从龟兹来，龟兹到长安三万余里，一路上可曾见过疏勒的太阳能板？”

罗石端茶的手顿了一下。

没有这个词。王猛说的是“太阳板”，是汉人晒毡用的芦席。全场的人都看着他。

他笑了一下，说疏勒人用毛毡晒酪，没有见过什么板。

回去后他在柴房里把那块折叠的太阳能板翻出来。板子表面全是划痕，折页处已经发白。电量早在两年前就归零了，最后那点电是给慕容垂那孩子退烧时开手电用的。那根充电线还在，线皮被老鼠啃掉一块，露出里面的铜丝。当年陆志强在准备区里捏着这根线说算不算一件，说手机和充电板已经算两样了，再加线就是第四样。

那是罗石最后一次在物理世界里和人拉扯一根线的归属。

线进了包。人进了舱。舱门锁了。

现在线留在了公元四世纪，发不了电，像一具尸体。

他把线塞回砖缝底下，上面压了半瓶 Costco 的布洛芬。瓶子已经黄了，标签上英文褪色，只能看清 Ibuprofen 200mg。还剩十七粒。数过很多遍了。瓶底用小刀刻了一道道杠，数数用的。

---

六月，蝉鸣刚起，前燕的背盟文书到了长安。

苻坚大怒。朝会上把那封帛书摔在地上，靴子踩上去。前燕杀了来使，说洛阳绝不让秦人踏入一步。

拜王猛为大都督，统内外诸军，伐燕。

调令第三天送到五重寺。

“着鸠摩罗石随军，为大军祈福。”

字是王猛亲笔。墨很浓，最后一笔带钩，像刀。

慕容垂府里的调令更早。慕容令为前锋，三千鲜卑突骑，拔洛阳。罗石听到调令时正敲木鱼，木槌砸在空木头上，一声闷响，空了。

笃。

他知道剧本走到哪一页了。

金刀计。

王猛要用慕容垂的金刀，调离慕容令，构陷慕容垂全家。这不是随机事件，是只读文件里写死的函数调用。调用栈已经在邺城的朝会和长安的假牙笑声里压好了。

罗石不能去找慕容垂。冠军将军府四周全是丞相府的眼线，每一个挑水的仆役，走路的姿势都不对。那是受过训练的步态识别。

他只能等慕容垂来寺里辞行。

那天黄昏，慕容垂来了，披着常服，眼睛里全是血丝。假牙的事后，他学乖了，在朝堂上和慕容评对骂得起劲，唾沫星子喷到殿柱上。他以为安全了。

“大师，随我出关吧，”他说，“洛阳一下，大燕就完了。可算了结我一桩心事。”

罗石没接话。他把慕容垂请进佛堂，关门，上闩，熄了灯，只留佛前一盏酥油灯。

从袈裟夹层里掏出一个锦囊。针线是他自己缝的，歪歪扭扭。

“王爷，”罗石声音压到只有气音，“昨夜佛祖托梦。说王爷有血光之灾。这是佛祖赐的八字真言，不到家破人亡的关头，千万莫拆。拆了，就不灵了。”

慕容垂一愣，手已经伸过来。锦囊很小，塞进掌心就没了。

他贴身藏好，点了点头，没问。

佛堂里木鱼又响了一声。

笃。

---

出征前夜，王猛去了冠军将军府。

罗石也在。名义上是替将军府诵经祈福，实则是王猛点名让他去的。

宴席摆在后院。酒是粟酒，淡。菜是几盘肉干。王猛喝得很慢，一杯酒分三次咽。

酒至酣处，王猛叹气：“此去关东，山高水长，生死难料。我与将军一见如故，恨不能为手足。”

他提出结拜。

慕容垂大笑说好。两人就在灯下歃血，换了庚帖。慕容垂是武人，动了真情，解下腰间那柄纯金打造的佩刀，双手递过去。

“此刀随我三十年，刀上铭文是我父燕王亲刻。景略若不嫌弃，权作信物。”

金刀在灯下晃眼。刀鞘是鲨鱼皮，刀柄嵌松石。罗石站在阶下，手里捧着经卷，看见王猛接刀时指尖微微发抖，不是激动，是用力握紧。

那年他二十一，第一次拥有这种不带刃的锋利。

王猛收刀入怀，又饮三杯，告辞。

慕容垂送到门口，回来时脚步都轻了。

罗石低头念经，没抬头。

笃。笃。

---

大军没有直扑洛阳。

王猛把中军藏进了邙山。邙山古道两边全是坟，几千年的坟，风一吹，纸钱打着旋。那是罗石当初坠落的地方，白骨半埋，乌鸦叫得空。

他在后军辎重队里，和几个伙夫一起推车。没人把一个祈福的胡僧当人。

前燕太傅慕容评得了细作急报，说王猛大军还在潼关集结，前锋只有慕容令孤军。于是急调五万精骑，星夜驰援洛阳。

五万人生生走进了布袋。

信号箭一响，山两侧伏兵四起。罗石推着车，听见山谷里先是闷雷一样的马蹄，然后是惨叫，像一锅水倒进了油里。燕军连洛阳城墙的影子都没见到，就在邙山坟堆间被包饺子了。

罗石没上去。他坐在坟边，掏出木鱼敲。

笃、笃、笃。

敲得很快，手心全是汗。尸体倒下去的声音像重物落水。

不到一个时辰，战事结束。五万燕军，全没了。洛阳守将站在城头看完全程，不等第二天，天没亮就开了城门，把印绶洗干净了捧出来。

慕容令的前锋已经渡了黄河。

王猛中军不动，给慕容令下令：继续东进，直取燕腹。

慕容令领命去了。

第二天夜里，罗石被叫到前锋营里做法事，说有夜惊。

他进帐时，看见一个浑身是血的人冲进来。披头散发，左臂耷拉着。

金熙。慕容垂在长安的心腹，替他喂马的。

“世子！世子！大事不好！”金熙扑在地上，从怀里掏出那柄金刀。金刀还在鞘里，血已经把鲨鱼皮染黑了，“王猛在长安构陷王爷！王爷已带家眷逃回燕国！这是王爷亲笔信，命你立刻脱离秦军，回邺城会合！”

信纸是熟的黄麻，字是慕容垂的笔迹，急，抖。

慕容令双手接刀，见刀如见人，刀刃上那道小缺口，是那年随父狩猎时崩的，做不得假。他瞬间红了眼，牙齿咬得咯咯响。

“景略小儿！欺人太甚！”

当夜，慕容令点三百亲兵，弃了辎重，仓皇向东，奔燕国去了。火把在黑地里连成一条断续的线很快不见。

罗石站在营帐外，手里的木鱼忘了敲。

金熙从帐里出来，经过他身边，低着头，没敢看他。身上血腥味里混着一点脂粉味，是长安才有的胡姬香。

罗石明白了。那不是血，是猪血。信是临摹的，刀是真的。人是买通的。

这是一个中间人攻击。Man in the middle。信任链被金刀劫持。

他想上去拽住慕容令，喊一声别走，那是伪造。那是 self-signed certificate。

但他不能。他的舌头在嘴里硬了一下，像旧伤。只要他喊出来，王猛的眼线下一秒就会把他也写进叛逃名单里。更重要的是，史书上慕容令这一逃，是写死的。

写死的意思是，只读。

只读文件不能写。写入就会触发系统自毁，就像当年赵婷在后台敲下的那串 rm -rf 和渐进式覆写密令。主控室的灯暗下去，数据库熔断，时间机器再也回不来。那是他自己选的。

他只能看着那条火线消失在黄河暮色里。

营地里风很大，黄沙打在脸上，像针。

---

几乎同一天，长安。

金熙日夜兼程，换了三匹马，赶回冠军将军府，连滚带爬跪在堂前。

“王爷！大事不好！世子慕容令在前线叛秦，已经投燕了！”

慕容垂眼前一黑，一把揪住他衣领：“你说什么？！”

前线军报比金熙只晚半个时辰到。王猛亲笔：慕容令不听调遣，率部潜逃投敌。

满门抄斩的罪。

慕容垂的手开始抖。第二个念头就是逃。现在就逃，趁长安城门还没关，带家眷奔河北。河北还有旧部。

家人哭成一片，收拾细软，马已备在后门。

混乱中，他的手碰到怀里。

硬的，小小一块。

锦囊。

万劫不复之死局，方可拆开。

他现在就是万劫不复。

他抖着手拆开，线头是罗石用牙齿咬断的，毛边。

里面一张黄纸，八个字。是罗石用不熟练的隶书写就：

**千万莫逃，入宫面圣。**

慕容垂盯着那八个字，看了很久。院子里人声嘈杂，孩子在哭。

他忽然把纸条攒紧，闭眼。

“都别收了。”他声音哑得像破风箱，“遣散家眷，各自去。我一人入宫。”

他换了白色素衣，解下佩剑，光着脚，赤着头，一个人，往皇宫走。

皇宫门前，卫士拦他。他跪下，说罪臣慕容垂求见天王。

---

太极殿前那块石砖是凉的。

慕容垂跪上去，膝盖骨生疼。他把假牙的事，金刀的事，儿子逃走的事，全说了，一字不漏，不敢添油加醋。

苻坚坐在龙椅上，听完，沉默。

殿里很静，能听见漏壶滴水。

苻坚站起来，慢慢走下御阶，走到慕容垂面前，伸手扶他。

“父子兄弟，罪不相及。”苻坚说，声音不高，“令儿去就去了。朕对卿之心，不改。”

他亲自替慕容垂披上外袍，当天降旨：进慕容垂为冠军将军，增邑。

从死罪到加官，只隔了一跪。

远在洛阳的王猛收到长安邸报，站在洛阳城头，看着邙山方向，一言不发，站了很久。风把他袖子吹得鼓起来。

他没输。慕容令回了邺城。

邺城那帮人，猜忌比刀快。听说慕容令是秦军前锋叛来，立刻认定是反间。还没等他进城，诏书就到了：流放。

死在流放路上。说是自缢，其实是军士用弓弦勒死的。草草埋在路边，连碑都没有。

消息传到前线，已是隆冬。

罗石正在帮军医分药。那瓶 Costco 布洛芬只剩十七粒了，他一粒一粒数着，分给发烧的兵。分到第三粒时，传令兵进来说，燕国那边，慕容令死了。

他手一抖，一粒白色的药片掉进泥里，瞬间脏了。

他弯腰去捡，捡起来，用袖子擦，擦不干净。

周围人还在说话，说大军就要开拔了，说邺城就要破了。

罗石捏着那粒脏了的药，忽然想起很久很久以前，赵婷在实验室里跟他说，开曼群岛那家空壳公司根本没有业务，那三千万不是什么高利贷，是洗钱。法人是姚雪梅弟弟，实际控制人是陆志强。

那时候他还问，充电线算不算一件行李。

现在药片在泥里，擦不干净。金刀在王猛手里，收不回来。慕容令在土里，扶不起来。

苻坚的宽恕是真的吗？是真的。宽恕一个人，比杀一个人难，这需要统治者有极大的 CPU 冗余去处理异常分支。这叫仁慈，也叫 bug。

而慕容令的死，也是真的。史书上写定的死，就是不可捕获的 exception。无论你在 catch 块里写多少锦囊，最终还是会抛出去。

罗石把那粒脏药扔了。

他掏出木鱼，坐在一根倒掉的辕木上，开始敲。

笃。

笃。

笃。

盛夏正午，烈日当空，他的后背却一点点凉了下去。从尾椎骨一直凉到后颈，像有一桶冰水贴着脊椎往里灌。

不是因为风。

是因为他终于摸到了那堵墙。

历史是一块只读存储器。ROM。锦囊能改的，只是内存里的一个临时变量，叫“慕容垂逃不逃”。而 ROM 里的那一行，写着“慕容令卒于流放道中”，谁都擦不掉。包括他这个从未来掉进来的野指针。

木鱼声在空旷的辕门外显得特别轻，却一下一下砸在石头上。

石旁添人，是佦。佦是一个人字旁加一个石。石头添了个人，石头就成了人。

他现在就是那块石头旁添的人。石头还在，人快不是人了。

他抬头，看见岁星在白天看不见的地方。他记得那年系统报废前最后一次气象模型推算里，写着今年岁星在吴，吴越将有大水。那天他在太极殿上说的“岁星分野”，一半是编的，一半是从模型日志里翻译出来的。

模型已经死了。药也快分完了。

而王猛的影子还长长地拖在邙山上没散。

罗石闭上眼，继续敲。敲给那个死在路边没有碑的年轻人，敲给那柄再也回不来的金刀，也敲给自己那瓶永远凑不齐五百粒的布洛芬。

笃。笃。笃。
